SOCIOSCAN

An AI-based system that helps identify eligible beneficiaries for government welfare schemes using data-driven analysis.

## 🚀 Features

* Smart eligibility prediction
* User-friendly interface
* Real-time data processing
* Secure and scalable design

## 🛠️ Tech Stack

* React (Vite)
* TypeScript
* Supabase
* Tailwind CSS

## 📦 Installation

```bash
npm install
npm run dev
```

## 📌 Usage

* Open the app in browser
* Enter required user details
* Get eligibility results instantly

## 🔐 Security

* Sensitive data is managed using environment variables
* `.env` files are excluded from version control
